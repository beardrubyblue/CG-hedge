# Инструкция по запуску бота автоматического хеджирования BTC на Hyperliquid

Этот бот предназначен для реализации стратегии Cash Secured Puts с автоматическим открытием SHORT позиции (хеджа) при падении цены до заданного уровня.

## Шаги по установке:

1. **Настройка окружения**:
   Скопируйте файл `.env.example` в новый файл `.env`:
   ```bash
   cp .env.example .env
   ```

2. **Заполнение данных в `.env`**:
   Откройте `.env` и введите свои данные:
   - `HL_WALET_ADDRESS`: Ваш публичный адрес кошелька на Hyperliquid.
   - `HL_PRIVATE_KEY`: Ваш приватный ключ (рекомендуется использовать API Agent Key).
   - `TELEGRAM_BOT_TOKEN`: Токен вашего бота от @BotFather.
   - `TELEGRAM_CHAT_ID`: Ваш ID в Telegram (можно узнать через @userinfobot).

3. **Запуск бота**:
   ```bash
   python3 main.py
   ```

## Команды бота:
- `/start` - Инструкция и текущие настройки.
- `/status` - Проверка текущей цены BTC и состояния мониторинга.
- `/set_price [X]` - Установить цену срабатывания (например, `/set_price 63000`).
- `/set_amount [USDC]` - Установить сумму для хеджирования.
- `/set_sl [Y]` - Установить смещение для Stop-Loss (например, `/set_sl 1000` создаст SL на уровне X + 1000).
- `/run` - **Запустить** процесс мониторинга цены.
- `/stop` - Остановить мониторинг.

## Логика работы:
- Бот мониторит цену BTC каждые 0.5 сек.
- При достижении уровня `TRIGGER_PRICE` выставляется Limit SHORT ордер.
- Одновременно выставляется Stop-Loss ордер на уровне `TRIGGER_PRICE + SL_OFFSET`.
- При открытии ордера и при срабатывании SL приходят уведомления в Telegram.
